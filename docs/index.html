Toggle

A simple toggle script for menus, accordions, navigation, and more.

What is Toggle?

Toggle is a lightweight JavaScript module that provides simple toggle interaction functionality using the accessible aria-expanded HTML attribute. Toggle has no dependencies, and can be easily implemented into any workflow or framework.

Getting Started

See Toggle on GitHub for installation and configuration instructions.

Basic Example

The most basic usage of Toggle transforms all targets with a class of toggle and adjacent <button> elements. The aria-expanded attribute can be used to control target visibility in your stylesheet.

Selecting Targets

Specify a custom target query selector to override the default .toggle selector. You can create as many instances of Toggle as you like.

Blur

Enable blur to collapse targets on click-away.

Class Names

Add class names to your toggle elements to provide more control over presentation in your stylesheet. Classes are available for toggle buttons, targets, and parents. We use the Font Awesome icon font in this example.

Accordions

Use the buttonSelector option to provide a custom query selector for toggle buttons. You can use this in conjunction with other configuration options to create a simple accordion or multi-level accordion navigation menu. We use the Font Awesome icon font in this example.

Create Buttons

Notice that in the previous example the accordion header itself is used as the toggle button and therefore is no longer clickable because the default click behavior is overriden by our Toggle script. Alternatively, we can use the createButtons option to automatically create a toggle button for each target found. This gives us the ability to create an accordion-style navigation menu with a separate clickable toggle button, allowing the accordion header link to remain clickable. We use the Font Awesome icon font in this example.

Dropdown Menus

Enable the blur and singleSibling options to create a horizontal dropdown navigation menu.

Off-Canvas Accordion Menu

Combine two toggles to create an off-canvas accordion menu. In this example, most of the work is done in our stylesheet.

Target Scope

If your toggle button is not adjacent to the toggle target in the HTML structure, you can use the scoped option to search the whole document for matches.
