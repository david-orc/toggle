Recipe: Accordion

Toggle isn't just for navigation. Configure the default query selector options and make use of the single and expandOnly configuration options to create a simple accordion.

In this example, our buttons are not adjacent to the toggle target in the HTML structure. Instead, we set buttonSelector to #{id}, which enables href attribute pairing to our toggle target id. This works just like a normal anchor tag in HTML (href="#some-id" is paired to id="some-id").

Script

var accordion = new Toggle({
  buttonClassExpanded: 'accordion-header-active',
  buttonSelector: '#{id}',
  expandOnly: true,
  single: true,
  targetSelector: '.accordion-panel'
})

CSS

/* Hide collapsed */
[aria-expanded="false"] {
  display: none;
}

/* Add icon to headers */
.accordion-header::after {
  content: '\f105'; /* Font Awesome 'angle-down' */
  display: inline-block;
  font-family: 'FontAwesome';
  margin-left: 0.5rem;
  transition: transform 0.3s ease;
}

/* Flip the icon */
.accordion-header-active::after {
  transform: rotate(90deg);
}
